1. If you don't have python installed, download the latest version here: https://www.python.org/downloads/

IMPORTANT: During the Python setup check "Add Python to Path"


2. The first time you install this program you need to run the setup.bat command. You only need to run this once. After the setup.bat is done it will close by itself.

3. Put the '.zip' or '.rar' file in the 'in' folder.

4. Run the 'auto_zip.pyw' to run Auto Unzip

5. Put all the .zip and .rar files in the 'in' directory. Press the extract button and all the extracted files will be put in the 'out' directory, in there own folders.
